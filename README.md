# Medical Insurance Cost Prediction

## Overview
This project predicts medical insurance costs based on features such as age, sex, BMI, number of children, smoking status, and region using Machine Learning regression techniques.

## Features
- Predict individual medical insurance costs
- Data preprocessing and cleaning
- Model training and evaluation
- Beginner-friendly ML project

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Pickle
